# TimeSeries

This folder is a workspace wrapper for the main BTCUSD forecasting / MLOps project located in `timeseries/`.

## Quickstart

```powershell
cd .\timeseries
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python run_project.py
```
